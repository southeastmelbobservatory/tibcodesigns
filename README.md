# Tibco Designs

Static modern website prototype for Tibco Designs, a Melbourne custom cabinetry business.

## Files

- `index.html` - one-page website
- `styles.css` - responsive styling
- `assets/` - downloaded Tibco Designs imagery used by the prototype

## Preview

Open `index.html` directly in a browser. No build step is required.

## Notes

This is a static front-end prototype. Contact links use `tel:` and `mailto:` links; there is no server-side form handling or CMS integration.
